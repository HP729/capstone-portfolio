Backend Development

These resources taught me how backend development functions. This includes creating APIs, handling HTTP requests, and interacting with the database using Express and Sequelize. I've used this to implement server routes, database queries, and communication between the frontend and backend.

  • ColorCode. Fetch API - JavaScript Tutorial for Beginners. 2023. YouTube, https://www.youtube.com/watch?v=ubw2hdQIl4E.
  • Model Querying - Basics | Sequelize. 25 Apr. 2025, https://sequelize.org/docs/v6/core-concepts/model-querying-basics/.
  • Model Querying - Finders | Sequelize. 25 Apr. 2025, https://sequelize.org/docs/v6/core-concepts/model-querying-finders/.
  • UPDATE Queries | Sequelize. 25 Apr. 2025, https://sequelize.org/docs/v7/querying/update/.
  • Using Express Middleware. https://expressjs.com/en/guide/using-middleware.html. Accessed 27 Jan. 2026.
